## WordPress My Photo Buckets Plugin
**Description**
* Designed a custom WordPress plugin using the WordPress API.
* Optimized the plugin for scalability with lazy loading and accessibility with WCAG 2 AA compliance.
* Integrated the fancybox.js library.
* Implemented CRUD (Create, Read, Update, Delete) functions for custom-styled photo albums. Seamlessly integrated with [WordPressâ€™s built-in gallery](https://developer.wordpress.org/themes/functionality/media/galleries/) creation feature.
